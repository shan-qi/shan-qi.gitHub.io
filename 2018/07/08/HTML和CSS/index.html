一、HTML语言基本介绍

1.基本结构

<html><!--根标签-->
    <head><!--网页头部（用于设计网页属性-->
        <meta charset="UTF-8"> <!--定义文本信息-->
        <title>哈哈哈 <!--网页的名字--></title>
    </head>
    <body>
        小琦    <!--主题部分(编辑区网页显示的内容部分)-->
    </body>
</html>

2.网页的基本标签元素

Tag Meaning
<h1></h1> 标题标签
<p></p> 段落标签
<em></em> 斜体标签
<br/> 换行标签
<hr/> 水平线标签
<strong></strong> 加粗标签
<blockquote></blockquote> 段落缩进
<font></font> 字体标签(color、size、face)
<center></center> 居中标签
<sup></sup> 上标
<sub></sub> 下标
<pre><pre> 原样输出

3.注释和特殊符号

特殊符号 注释
&nbsp; 空格(space)
&gt; 大于号(>)
&lt; 小于号(<)
&copy; 版权所有(©️)
&reg; 注册商标(®️)

4.超链接

1)在同一页面下:

(1)打锚点:<a name="锚点名称"></a>
(2)创建跳转链接:<a href="锚点名称">开始跳转</a>

2)在不同的页面下:

(1)打锚点:<a name="锚点名称">开始跳转</a>
(2)创建 跳转链接:<a href="资源文件或者资源地址#锚点名称">开始跳转</a>

5.图片标签

<img src="图片路径" alt="图片替代文本" title="鼠标悬浮文字" width="图片宽度" height="图片高度"/>

6.列表标签

无序号:
    <ul type="circle">
        <li>小琦</li>
        <li>小猪</li>
    </ul>
有序号:        
    <ol type="circle">
         <li>小琦</li>
        <li>小猪</li>
    </ol>

7.块标签和行内标签

<div>块标签</div>

<span>行标签</span>

8.表格标签

border || align || width || height

9.表单标签

<form action = "#" method = "get/post"></form>

form标签属性:

1) action:整个表单提交的位置(可以使一个页面，也可以是JAVA代码)

2) method:表单提交的方式(get/post/delete…)

post
<ul type="circle">
        <li>不会将信息展示到地址栏</li>
        <li>提交文件大小无限制</li>        
</ul>
get
<ul type="circle">
        <li>将用户信息展示到地址栏</li>
        <li>提交文件大小无限制</li>        
</ul>

二、CSS介绍

1.CSS指层叠样式表。
2.样式定义如何显示HTML元素。
3.样式通常存储在样式表中。
4.把样式添加到HTML4.0中是为了解决内容与表现分离问题。
5.外部样式表可以极大的提高工作效率。
6.外部样式表通常存储在CSS文件中。
7.多个样式表可以层叠。

10.CSS的三种引入方式

  • 第一种:行内引入
    <div style="color: red; font-size: 100px;">程序猿</div>
  • 第二种:内部引入
    <style type="text/css"> .box{ color:red; font-size: 100px; } </style> <div class="box">程序猿</div>
  • 第三种:外部引入
    <link rel="stylesheet" type="text/css" herf="../css/style.css">

11.CSS选择器

CSS选择器有三种(元素选择器、类选择器、id选择器)

元素选择器
元素名{
        属性名：属性值;
}

id选择器
#id属性名{
属性名：属性值;
}        

类选择器
.类选择器{
属性名：属性值;
}

12.CSS常用的属性

(1)CSS文本 

color: 
line-height:  
letter-spacing: 
text-align: 
word-spacing:

(2)CSS字体 

font-family:字体系列(类型)  
font-size:字体尺寸
font-style:字体的样式
                normal:标准的字体样式
                italic:斜体
   font-wight:设置字体的粗细  
   CSS字体的简写属性: font:italic bold 36px "黑体" ;

(3)CSS列表

list-style-type:设置列表项前面的类型
list-style-image：常用的属性值:URL 图像的路径

(4)CSS背景

background-image:
background-repeat:
background-position:
background-color:

(5)CSS表格

border-collapse :collapse;  合并边框

(6)CSS边框

border-style（上、右、下、左）
border-width（上、右、下、左）
border-color（上、右、下、左）

13.CSS让块元素成为内联元素

display: inline-block

14.div居中

margin: 0 auto;

15.块元素内容居中

text-align: center;
