###二.CSS介绍 

1.CSS指层叠样式表。
2.样式定义如何显示HTML元素。
3.样式通常存储在样式表中。
4.把样式添加到HTML4.0中是为了解决内容与表现分离问题。
5.外部样式表可以极大的提高工作效率。
6.外部样式表通常存储在CSS文件中。
7.多个样式表可以层叠。

####10.CSS的三种引入方式

  • 第一种:行内引入
    <div style="color: red; font-size: 100px;">小琦</div>
  • 第二种:内部引入
    <style type="text/css"> .box{ color:red; font-size: 100px; } </style> <div class="box">JavaEE</div>
  • 第三种:外部引入
    <link rel="stylesheet" type="text/css" herf="../css/style.css">

####11.CSS选择器
CSS选择器有三种(元素选择器、类选择器、id选择器)

元素选择器
元素名{
        属性名：属性值;
}

id选择器
#id属性名{
属性名：属性值;
}        

类选择器
.类选择器{
属性名：属性值;
}
